# Tic-Tac-Toe
A two-player Tic-Tac-Toe game built with vanilla HTML, CSS, and JavaScript. Features a retro neon arcade UI with glowing symbols, animated win detection, and live score tracking.

# Live Demo
https://kaintura-priyanshu.github.io/Tic-Tac-Toe/

# Features
- Two-player turn-based gameplay (X vs O)
- Win detection across all 8 combinations
- Animated golden win line on victory
- Live scoreboard with round and full reset
- Retro neon arcade design — no frameworks, no dependencies
- Fully responsive — works on mobile and desktop

# Run Locally
No setup needed. Just open `index.html` in any browser.

# Built With
- HTML5
- CSS3 (custom properties, animations, CSS grid)
- Vanilla JavaScript (ES6+)
- Google Fonts — Orbitron & Share Tech Mono
